# Nightreign Boss Roster

A small web tool to visualize boss combinations in **ELDEN RING: NIGHTREIGN**.

Select a **Day 1 boss**, **Day 2 boss**, or **Nightlord**, and the page will highlight all valid combinations.

## GitHub Pages  

[hushade.github.io/nightreign-boss-roster](https://hushade.github.io/nightreign-boss-roster/)

## Features

- One-screen boss combination viewer
- Multidirectional filtering between Day1 / Day2 / Nightlord
- Fixed card layout (no reflow while filtering)
- Mobile friendly layout

## Data Structure

Data is separated into two JSON files.
